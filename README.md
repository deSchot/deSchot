### 🔦 Egress Enabler — live tool

A browser-based egressibility assessor: a rebuild of the Smedberg et al. (2023) instrument. Mark each environmental barrier compliant or non-compliant and watch the Egress Enabler Score recompute live across all 14 functional limitations, then switch to Impact mode to see how design changes move egressibility against a saved baseline.

**▶️ [Launch the tool](https://deschot.github.io/EgressEnabler/)** · [Source & docs](https://github.com/deSchot/EgressEnabler)
